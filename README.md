♻️ Recycle.AI Discord Bot

Bot de Discord que analiza imágenes de objetos y te dice cómo reciclarlos correctamente usando inteligencia artificial (OpenAI GPT-4o).

🚀 Características
📸 Analiza imágenes enviadas en Discord
🧠 Detecta objetos automáticamente con IA
♻️ Indica cómo reciclar cada material
🌍 Consejos ecológicos
⚡ Respuestas rápidas en el servidor
🛠️ Tecnologías usadas
Python 3.9+
discord.py
OpenAI API (GPT-4o-mini con visión)
python-dotenv
requests
